# 貢獻指南

## Certainty Tag Language

This project uses **中文** certainty tags.
<!-- Options: English | 中文 -->

### 標籤參考

| 標籤 | 英文對照 | 使用時機 |
|------|----------|----------|
| [已確認] | [Confirmed] | 直接從程式碼/文件中取得的證據 |
| [推論] | [Inferred] | 從證據合理推導出的結論 |
| [假設] | [Assumption] | 基於常見模式的猜測，需驗證 |
| [未知] | [Unknown] | 目前無法取得的資訊 |
| [需確認] | [Need Confirmation] | 需要使用者來釐清的問題 |

### 衍生標籤（用於生成新內容）

| 標籤 | 英文對照 | 使用時機 |
|------|----------|----------|
| [來源] | [Source] | 直接來自規格/需求的內容 |
| [衍生] | [Derived] | 從來源內容轉換而來 |
| [生成] | [Generated] | AI 產生的結構 |
| [待辦] | [TODO] | 需要人工實作 |
